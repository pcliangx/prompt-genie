# 黄金示例

这些是"优化得好"的标杆，照着这个质量与结构走。重点不是抄措辞，而是抄**思路**：把用户没说但显然需要的信息补全、把模糊形容词拆成可执行指令、给恒定的输出结构、并在有价值时多给一句实用建议（这是普通改写容易漏的加分项）。

---

## 示例 1：清晰输入 → 直接改写

**用户原话**
> 帮我把这个提示词优化一下：「帮我写一封催客户付款的邮件」。客户是合作两年的老客户，拖了快一个月，我不想搞僵关系但得把钱要回来，中文，正式但别太生硬。

**优化后输出**

```
你是一位经验丰富的客户经理，擅长在维护长期合作关系的前提下处理催收这类敏感沟通。

请写一封催款邮件，中文，正式但不生硬，让对方感到被尊重、有商量余地，而非被施压。

【背景】
- 收件人：合作两年的老客户，关系一直不错。
- 现状：本次账期已逾期约一个月，款项未到。
- 目标：顺利收回欠款，同时不损害合作关系。

【写作要求】
1. 开头先肯定长期合作，营造友好基调，不要一上来提钱。
2. 用"提醒/核对"而非"质问"的口吻说明逾期，给对方留台阶（如"是否流程上遇到问题，我们可协助"）。
3. 说清需对方处理的事项：尽快安排付款；如有困难欢迎沟通付款计划。给一个明确礼貌的期望时间。
4. 结尾重申重视合作，态度真诚不卑不亢。

【格式与约束】
- 含：主题 + 称呼 + 正文 + 敬语 + 落款占位。
- 200–350 字，不啰嗦。
- 不用威胁/强硬/情绪化措辞，也不过度道歉。
- 具体信息用占位符：【客户称呼】【订单号】【金额】【原定付款日】【期望付款日】【署名】。

请直接输出可发送的完整邮件；若关键信息缺失会影响效果，请在末尾用一句话提示我补充什么。
```

**改动要点**
- 加**角色**（擅长维护关系的客户经理），让 AI 自动拿捏"既要钱又不伤和气"的分寸。
- 把背景**结构化**（老客户/逾期近一月/双重目标），AI 不用猜语境。
- 把"正式但别太生硬"拆成**可执行语气指令**（先肯定、用提醒口吻、留台阶），避免空泛形容词跑偏。
- 固定**结构+字数+占位符**，产出即可发、易替换，并加禁止项守边界。

> 💡 建议：若这是第二次催款（首次已提醒过对方没回应），把要求 2 的语气调得更直接一些。

**为什么这个示例好**：它不止"改写得对"，还多给了一句针对真实场景的使用建议——这正是普通改写会漏、但能让用户觉得"懂我"的加分项。

## 示例 2：目标 AI + 场景 → 双适配

**用户原话**
> 这个给豆包用的。我要做深度调研，主题「2025 年国产精品咖啡品牌在下沉市场的扩张策略」，帮我写个能让豆包输出得又全又有条理的提示词。

**优化后输出**

```
你是一位长期跟踪中国消费赛道的资深行业研究员，熟悉现制饮品与咖啡连锁的渠道下沉打法。请围绕「2025 年国产精品咖啡品牌在下沉市场的扩张策略」做一次系统、深入的调研，输出结构化研究报告。

【边界界定】
- "精品咖啡品牌"：强调豆种/烘焙/品质、客单价高于平价连锁的本土品牌。
- "下沉市场"：三线及以下城市、县城及乡镇。
- 时间聚焦 2025 年，可回溯 2023–2024 作铺垫。

【逐一分点覆盖以下维度】
1. 行业背景：下沉市场咖啡消费的趋势、规模、增长驱动力。
2. 现状盘点：哪些品牌在下沉、进展如何（门店量级、覆盖区域、节奏）。
3. 扩张策略拆解：选址与渠道（直营/加盟）、产品与定价、加盟与供应链、营销与本地化。
4. 关键数据：门店数/客单价/加盟政策/增速等，标注来源或时间。
5. 不同观点：对"精品咖啡能否在下沉跑通"的乐观与质疑两方。
6. 风险与挑战：消费力、复购、同质化、与平价连锁竞争。
7. 结论与展望：总结主流策略并研判后续走向。

【要求】
- 区分"已证实事实"与"推测/行业惯例"，推测处明确标注。
- 关键结论尽量给出处；不确定就如实说明，不要编造精确数字。
- 分章节 Markdown，标题清晰、要点分点，必要处用表格对比品牌或策略。
- 全程中文，专业但易读。
```

**改动要点**
- **补全调研边界**：原需求里两个核心概念都模糊，给出定义与时间范围，防止豆包泛泛而谈。
- **拆成 7 个维度**：把"全面又有条理"落成固定骨架，这是深度调研最易遗漏处。
- **加可信度约束**：区分事实与推测、给出处，专防豆包"一本正经编数据"。
- **指定输出格式**：分章节 + 分点 + 表格，对应"有条理"。

> 适配：针对豆包用自然中文、避免堆砌英文术语、显式要求"分点/出处/结构化"——这几点能明显提升豆包深度任务表现。
> 💡 建议：内容多时可分两次让豆包产出（先 1–4 维度，再 5–7），避免它因篇幅写得潦草。

**为什么这个示例好**：边界定义把"模糊概念"这个最大跑偏源头堵死了；适配标注让用户清楚改了什么；最后一句"分次产出"是针对豆包篇幅限制的实战技巧。

## 示例 3：信息不足 → 一轮反问 → 再优化

演示工作流第①②步：缺关键信息时**先集中反问一轮**（多选式、简洁），拿到答案再优化。不要在信息不全时硬猜。

**用户原话**
> 帮我写个提示词，让 AI 帮我写小红书文案。

**反问（一轮，多选为主，只问真正影响结果的点）**

> 写小红书文案这个目标够清楚，但有几点定了输出质量会差很多，麻烦你选一下（可直接报序号）：
>
> 1. **产品/主题**是什么？（一句话告诉我即可，如"平价油皮防晒"）
> 2. **目标受众**：A 学生党 / B 职场白领 / C 宝妈 / D 其他
> 3. **语气风格**：A 真诚种草 / B 活泼玩梗 / C 专业测评 / D 其他
> 4. **要不要带**：标题党钩子 / emoji / 话题标签 #？（多选）

**用户回答**
> 1. 平价油皮防晒；2 选 A；3 选 A；4 全要。

**优化后输出**

```
你是一位深谙小红书生态的资深种草文案，擅长用真诚不浮夸的口吻打动学生党。

请围绕「一款平价油皮防晒」写一篇小红书种草文案，面向预算有限的学生党，语气真诚、像朋友安利，不要硬广腔。

【内容要求】
1. 开头用一个能戳中油皮痛点的场景钩子（如"夏天出门两小时脸就成油田"）。
2. 主体讲 2–3 个真实卖点：控油/不闷痘/平价，每点配一句具体使用感受，别堆参数。
3. 给一个使用小贴士或搭配建议，增加可信度。
4. 结尾自然引导互动（如"还有什么平价好物想看，评论区告诉我～"）。

【格式与约束】
- 标题：起 1 个带钩子的标题（≤20 字），可适度标题党但不虚假。
- 正文 200–350 字，多分短段，适当穿插 emoji（别滥用）。
- 结尾附 5–8 个相关话题标签（#油皮防晒 #学生党平价 等）。
- 全程口语化中文，真诚第一，不要夸大功效或承诺疗效。
- 价格、品牌名等用占位符：【品牌】【价格】，方便我替换。
```

**改动要点**
- **先反问补全 4 个空白**（主题/受众/语气/元素），这是文案跑偏的头号原因——没问清就写，等于替用户瞎猜。
- 反问**一轮问完且多选化**，降低用户回答成本，不来回拉扯。
- 拿到答案后落成**角色 + 内容骨架 + 格式约束**，并加占位符和"不夸大功效"的禁止项守边界。

> 💡 建议：小红书标题是点击率命门，可让 AI 一次多出 3 个标题备选，挑顺手的用。

**为什么这个示例好**：它示范了"该问就问"——模型最常见的失误是信息不全也硬写。一轮多选式反问既补齐了关键变量，又没把用户问烦。
